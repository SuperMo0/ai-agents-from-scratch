# ai-agents-from-scratch

A reference repo for building AI agents from scratch — no agent frameworks
(LangChain, LangGraph, PydanticAI), just direct API calls and explicit
control flow. Each subdirectory is a self-contained lesson/exercise.

## Lessons

- [agent101](agent101/) — a research agent with tool use, web search, and an
  automatic evaluator gate that checks answer quality before finishing.
